## About

2024_SESAC_FastAPI_특강

FastAPI를 활용한 AI 모델 서빙 및 배포

### Step
* [FastAPI](!https://fastapi.tiangolo.com/) installation
```
pip install fastapi
pip install uvicorn
```
* Run Server
```
uvicorn main:app --reload
```
